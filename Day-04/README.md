# Day 4 - Arrays

# Problems Solved

## 1. Remove Duplicate Elements (Sorted Array)

Remove duplicate elements from a sorted array so that each element appears only once while maintaining the sorted order.

**Note:** Since the array is already sorted, duplicate elements are adjacent, making it easier to identify and remove them.

## 2. Find Maximum Difference

Find the maximum difference between two elements in an array.

**Note:** If no such pair exists, the maximum difference can be `0` or a negative value, depending on the problem statement.

## What I Learned

* How to work with sorted arrays efficiently.
* How to identify and remove duplicate elements.
* How to compare elements to find the maximum difference.
* Improved understanding of array traversal and problem-solving techniques.
