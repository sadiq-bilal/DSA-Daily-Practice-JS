# Day 5 - Arrays

# Problems Solved

## 1. Find an Element Without Using Its Index

Find a specific element in an array by traversing it and comparing each element until the required value is found.

**Note:** Traverse the array sequentially without directly accessing a known index.

## 2. Move All Zeros to the End

Move all `0` elements to the end of the array while maintaining the relative order of the non-zero elements.

**Note:** Rearrange the array without changing the order of non-zero elements.
