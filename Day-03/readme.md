# Problems Solved

## 1. Count Even and Odd Numbers in an Array

Count the total number of even and odd elements in an array by checking each element and updating the respective count.

## 2. Find Second Largest Element in an Array

Find the second largest distinct element in an array by traversing the array and keeping track of both the largest and second largest values.


## What I Learned

* How to find the second largest element in an array.
* How to use multiple variables while traversing an array.
* How to identify even and odd numbers using the modulus (`%`) operator.
