# Day 1 - Arrays

# Problems Solved

## 1. Largest Element in an Array
Find the largest element by traversing the array and updating the maximum value whenever a larger element is found.

## 2. Smallest Element in an Array
Find the smallest element by traversing the array and updating the minimum value whenever a smaller element is found.

## What I Learned

- How to access array elements.
- How to traverse an array.
- How to compare elements to find the largest and smallest values.
