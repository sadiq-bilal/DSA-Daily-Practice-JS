# Day 2 - Arrays

# Problems Solved

## 1. Average of an Array

Calculate the average of all elements in an array by finding the sum of the elements and dividing it by the total number of elements.

## 2. Reverse an Array

Reverse the elements of an array by swapping the first element with the last, the second with the second last, and so on until the middle of the array is reached.

## What I Learned

* How to calculate the sum of array elements.
* How to find the average of an array.
* How to reverse an array using element swapping.
* Improved understanding of array traversal and indexing.
